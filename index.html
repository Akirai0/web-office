MikroTik CCR2004-16G-2S+

CCR2004-16G-2S+ เป็นเราท์เตอร์ประสิทธิภาพสูง มาพร้อมพอร์ต 16x Gigabit Ethernet และ 2x 10G SFP+ รองรับการใช้งานองค์กรขนาดกลางถึงใหญ่ ด้วย CPU 4-core 1.8GHz ประมวลผลได้รวดเร็ว รองรับการจัดการเครือข่ายขั้นสูง

MikroTik CRS328-24P-4S+RM

CRS328-24P-4S+RM สวิตช์ PoE+ ระดับองค์กร มาพร้อม 24 พอร์ต Gigabit Ethernet PoE-out และ 4 พอร์ต 10G SFP+ รองรับ PoE+ 802.3at/af จ่ายไฟรวมสูงสุด 450W เหมาะสำหรับติดตั้งกล้อง IP และ Access Point

Huawei eKitEngine AP362

AP362 เป็น WiFi 6 Access Point รุ่นใหม่ล่าสุดจาก Huawei รองรับ 4x4 MU-MIMO ความเร็วสูงสุด 5.4 Gbps พร้อมระบบ AI-powered optimization ช่วยปรับแต่งสัญญาณอัตโนมัติ เหมาะกับองค์กรที่ต้องการประสิทธิภาพสูง

Synology DiskStation DS923+

NAS 4 ช่อง รองรับ M.2 NVMe SSD cache มาพร้อม CPU AMD Ryzen และ RAM 4GB (อัพเกรดได้ถึง 32GB) สามารถขยายความจุได้ถึง 72TB พร้อมฟีเจอร์ Synology Photos, Surveillance, File Sharing และอื่นๆ
